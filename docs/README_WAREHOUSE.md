# Модуль «Управление товарами на складе»

## Запуск
1. Откройте MySQL Workbench и выполните `database/warehouse_schema_and_seed.sql`.
2. В `EpsteinsMarket/App.config` укажите логин и пароль MySQL в строке `WarehouseDBEntities`.
3. Откройте `EpsteinsMarket/WarehouseManagement.csproj` в Visual Studio, восстановите NuGet-пакеты и запустите проект.

Тестовые пользователи: `admin/admin`, `store/store`, `manager/manager`.

## Структура БД
База `warehouse_management` содержит роли, сотрудников, пользователей, категории, единицы измерения, поставщиков, клиентов, склады, зоны, места хранения, товары, остатки, поступления с позициями, отгрузки с позициями, инвентаризации с результатами и журнал действий.

## Реализовано
Авторизация, роли, журналирование входа и операций, просмотр/поиск товаров, CRUD товаров, просмотр остатков, администрирование пользователей, просмотр отгрузок, сервисные методы для поступлений, отгрузок с проверкой остатка и инвентаризации.

## Принятые решения
Проект сохранён как WPF/.NET Framework с Entity Framework 6. Подключение переведено на MySQL через конфигурацию. Старые страницы интернет-магазина переиспользованы как складские экраны, чтобы сохранить стиль и навигацию исходного приложения.
